HTML has two types of elements : In-line elements (span), block elements (div).

In HTML, page layout refers to the arrangement of elements such as headings, paragraphs, images, forms, and other content within a web page. A well-designed page layout is essential for creating an engaging user experience, improving readability, and enhancing overall visual appeal. In HTML, page layout is achieved through the use of **containers** like `div` and `section` elements, which group related content together. These containers can be styled using CSS to control their dimensions, padding, margin, border, and background color. Additionally, HTML provides various semantic elements like `header`, `nav`, `main`, `aside`, and `footer` that define the structure of a web page and provide a clear hierarchy of content. By combining these structural elements with CSS styling, developers can create visually appealing and user-friendly page layouts that meet the needs of diverse audiences.
